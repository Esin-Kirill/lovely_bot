# Lovely bot

## Описание
Телеграмм-бот, написанный с помощью библиотеки pyTelegramBotAPI. Писал из интереса, для своей девушки.
Бот имеет небольшой функционал, но главный - это отправка различных вариантов сообщений "с добрым утром". В дальнейшем планирую его немного расширить:
1. Добавить изучение английского языка.
2. Поддержание несложного диалога.
3. Одну-две мини-игры, чтобы скоротать время (например, пока едешь на работу).

Запущен на платформе Heroku.
